# Feature Specification: 初期ピン配置ガイド

**Feature Branch**: `015-pin-setup-guide`
**Created**: 2026-03-03
**Status**: Draft
**Input**: User description: "初期ピン配置ガイド: ゲーム開始前（またはゲーム中に参照可能な形で）、モルックのピン配置図をビジュアル表示する。通常ゲームでは4行三角配置（1-2 / 3-10-4 / 5-11-12-6 / 7-9-8）、Mölkkoutでは5ピン直線配置（6-4-12-10-8）を表示する。ルールを知っているプレイヤーが素早く配置確認できることを重視。"

---

## User Scenarios & Testing *(mandatory)*

### User Story 1 - 通常ゲーム開始前のピン配置確認 (Priority: P1)

屋外でモルックを始めようとしているプレイヤーが、ピンを並べながらアプリで「どのピンをどこに置くか」をさっと確認したい。ゲームセットアップ直後（ゲーム画面遷移後）に、ゲーム画面から配置ガイドを即座に参照できる。

**Why this priority**: ピン配置は毎ゲーム発生する最初のステップ。この機能だけで「セットアップ時に迷わず配置できる」という核心的な価値が実現する。

**Independent Test**: セットアップ完了後のゲーム画面から配置ガイドを開き、4行三角配置（1-2 / 3-10-4 / 5-11-12-6 / 7-9-8）と 3.5m の投球距離が表示されることを確認することで単独テストできる。

**Acceptance Scenarios**:

1. **Given** ゲーム画面が表示されている状態で、**When** プレイヤーが配置ガイドボタンをタップする、**Then** 4行三角配置（Row1: 1・2、Row2: 3・10・4、Row3: 5・11・12・6、Row4: 7・9・8）のビジュアル図と投球ラインからの距離（3.5m）を示したモーダルが表示される
2. **Given** 配置ガイドモーダルが表示されている状態で、**When** プレイヤーが閉じるボタンまたはモーダル外をタップする、**Then** モーダルが閉じてゲーム画面に戻り、ゲーム状態（スコア・ターン）は変化しない
3. **Given** 配置ガイドモーダルが表示されている状態で、**When** プレイヤーが何も操作しない、**Then** ゲームのターンは進まず、モーダルを閉じるまでゲームは中断状態を維持する

---

### User Story 2 - Mölkkoutのピン配置確認 (Priority: P2)

Mölkkoutゲーム中に、5ピンの直線配置（6-4-12-10-8）と各ピン間の距離を確認したい。Mölkkoutでは通常ゲームと異なる専用配置なので、混乱を避けるため専用の表示が必要。

**Why this priority**: Mölkkoutは頻度が低いため P2 とする。ただしMölkkoutプレイヤーにとっては通常ゲームと同様に重要な情報であり、既存のMölkkoutゲーム画面への追加で実現できる。

**Independent Test**: Mölkkoutゲーム画面から配置ガイドを開き、5ピン直線配置（左から #6・#4・#12・#10・#8）とピン間隔（1 Mölkky スティック分）が表示されることを確認することで単独テストできる。

**Acceptance Scenarios**:

1. **Given** Mölkkoutゲーム画面が表示されている状態で、**When** プレイヤーが配置ガイドボタンをタップする、**Then** 5ピン直線配置（左から #6・#4・#12・#10・#8）、投球ラインから先頭ピンまでの距離（3.5m）、ピン間隔（1 Mölkky スティック分）を示したモーダルが表示される
2. **Given** Mölkkoutゲーム画面の配置ガイドモーダルが表示されている状態で、**When** プレイヤーがモーダルの見出しと配置図を確認する、**Then** Mölkkout専用の配置であることが見出し等で明確に区別されており、通常ゲームの三角配置と混同しない
3. **Given** Mölkkoutゲームが進行中で配置ガイドを開く場面で、**When** プレイヤーが内容を確認する、**Then** 「倒れたピンは元の位置に戻す」というMölkkout固有のルール注記がガイド内に表示される

---

### User Story 3 - ゲーム中の配置再確認 (Priority: P3)

ゲームが進行してピンが散らばった後も、初期配置の記憶が曖昧なプレイヤーが元の配置を確認したい（戦略的判断やルール確認のため）。

**Why this priority**: P1・P2 の配置ガイドボタンが実装されれば、ゲーム中いつでも参照できるため追加実装は不要。このユーザーストーリーは P1・P2 の自然な恩恵として自動的に実現される。

**Independent Test**: ゲーム開始後に複数ターン進んだ状態でも配置ガイドボタンが表示され続け、タップすると正しい初期配置図が表示されることで確認できる。

**Acceptance Scenarios**:

1. **Given** ゲームが数ターン進行した状態で、**When** 配置ガイドボタンをタップする、**Then** ゲーム開始時の初期配置図が表示される（現在の散らばった状態ではない）
2. **Given** 配置ガイドモーダルが表示されている状態で、**When** プレイヤーが内容を確認する、**Then** ガイドは「初期セットアップ」の参照であることが明示されている

---

### Edge Cases

- 配置ガイドが開いている間にゲームのターンが意図せず進まないか？ → モーダルが開いている間はゲーム入力操作がブロックされる
- 言語切り替え後に配置ガイドを開いた場合、ラベルや注記が正しく翻訳されて表示されるか？ → ガイド内テキストは i18n 対応（日本語・英語・フィンランド語）
- 非常に小さい画面でもピン番号や配置図が読み取れるか？ → モーダル内はスクロール可能にし、最小フォントサイズを確保する
- 横向き（ランドスケープ）モードでのモーダル表示は？ → 縦向きと同じフルモーダルを使用し、ランドスケープ専用のレイアウトは設けない

---

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: システムは通常ゲーム画面の `ScreenHeader`（バックボタンと同じヘッダーバーの右側）に、配置ガイドを開くためのアイコンボタンを常時表示しなければならない（MUST）
- **FR-002**: システムは Mölkkout ゲーム画面の `ScreenHeader`（バックボタンと同じヘッダーバーの右側）に、配置ガイドを開くためのアイコンボタンを常時表示しなければならない（MUST）
- **FR-003**: 通常ゲームの配置ガイドは、12ピンの4行三角配置（Row1: 1・2 / Row2: 3・10・4 / Row3: 5・11・12・6 / Row4: 7・9・8）を視覚的に表示しなければならない（MUST）。各ピンには番号を明示する
- **FR-004**: 通常ゲームの配置ガイドは、投球ラインから最前列ピンまでの距離（3.5m）を表示しなければならない（MUST）
- **FR-005**: Mölkkout の配置ガイドは、5ピンの直線配置（左から #6・#4・#12・#10・#8）を視覚的に表示しなければならない（MUST）。各ピンには番号を明示する
- **FR-006**: Mölkkout の配置ガイドは、投球ラインから先頭ピンまでの距離（3.5m）とピン間隔（1 Mölkky スティック分）を表示しなければならない（MUST）
- **FR-007**: Mölkkout の配置ガイドは「倒れたピンは元の位置に戻す」というMölkkout固有のルール注記を含まなければならない（MUST）
- **FR-008**: 配置ガイドはモーダルとして表示し、閉じるボタンまたはモーダル外タップで閉じられなければならない（MUST）。モーダルが開いている間はゲーム入力をブロックする
- **FR-009**: 配置ガイド内のすべてのテキストは i18n 対応（日本語・英語・フィンランド語）でなければならない（MUST）
- **FR-010**: 配置ガイドの内容はゲームのターン進行状態に関係なく、常に「初期セットアップ時の配置」を表示しなければならない（MUST）

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: プレイヤーが配置ガイドボタンをタップしてから配置図が画面に表示されるまでの操作が1タップで完了できる
- **SC-002**: 配置ガイドを閉じた後、ゲームのスコア・ターン状態が配置ガイドを開く前と100%一致している（ゲーム状態への副作用ゼロ）
- **SC-003**: モルックのルールを知っているプレイヤーが、配置ガイドを参照してピンのセットアップを完了するのに30秒以内で済む
- **SC-004**: 通常ゲームとMölkkoutの配置ガイドを見比べたとき、プレイヤーが「これは別の配置だ」と誤解なく認識できる（誤配置防止）

## Clarifications

### Session 2026-03-03

- Q: ゲーム開始時に配置ガイドを自動表示するか？ → A: オンデマンドのみ（ボタンタップで開く）。ゲーム開始時の自動表示はしない。
- Q: 配置ガイドボタンの配置場所は？ → A: ScreenHeader に統合（バックボタンと同じヘッダーバーの右側にアイコン追加）
- Q: ランドスケープモードでのモーダル表示は？ → A: 縦向きと同じフルモーダル（ランドスケープ専用の特別対応なし）

---

## Assumptions *(optional)*

- **A-001**: 配置ガイドはルールを「知っている」プレイヤー向けのクイックリファレンスであり、初めてのプレイヤーへのルール説明機能は対象外とする
- **A-002**: 配置図の表現はシンプルなテキストベースのビジュアル（各ピンを丸や四角で表現し番号を記載する形式）とし、画像ファイルは使用しない
- **A-003**: 配置ガイドボタンは `ScreenHeader` に統合し、ヘッダーバーの右側（バックボタンの反対側）に配置する。ゲーム開始時の自動表示はせず、プレイヤーがボタンをタップしたときのみ表示する（オンデマンド）
- **A-004**: ホーム画面・セットアップ画面には配置ガイドを設けない（通常ゲーム画面・Mölkkoutゲーム画面からのみアクセス可能）
